Open-source few-shot prompt studio for building, testing, and managing few-shot examples.

## Quick Start

```bash
# Clone the repo
git clone https://github.com/intelligent-iterations/few-shot-studio.git
cd few-shot-studio

# Install dependencies
pnpm install

# Run checks
pnpm check
pnpm test
pnpm build
```

## Development

```bash
# Type-check
pnpm check

# Lint
pnpm lint

# Format
pnpm format

# Test with watch mode
pnpm test:watch

# Test with coverage
pnpm test:coverage

# Build
pnpm build
```
